# Responsive Card Slider

![Preview](img/Preview.jpg)

A simple responsive card slider built with HTML, CSS, and JavaScript. This project demonstrates a clean, touch-friendly card slider UI that adapts to different screen sizes.

**Preview:** A quick visual of the slider is shown above.

**Features:**

- Smooth card sliding animation
- Responsive layout for mobile and desktop
- Keyboard and touch support (if available in browser)
- Lightweight: no external dependencies

**Getting Started**

1. Clone or download the repository.
2. Open the `index.html` file in your browser.

For a quick local preview, you can run a simple static server (optional):

```bash
# Using Python 3
python -m http.server 8000
# Then open http://localhost:8000 in your browser
```

**Project Structure**

- `index.html` — main HTML file
- `Style.css` — project styles
- `javaScript.js` — slider behaviour and interactions
- `img/` — images used by the project (includes `Preview.jpg`)
